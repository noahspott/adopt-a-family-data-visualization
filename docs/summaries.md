### 1. Adopt-A-Family
Adopt-A-Family is a 19-year-old charitable program sponsored by the Cathedral of Christ the King in Atlanta that bridges donors with families in need. The Christmas program aims to serve over 750 families by providing gifts, basic necessities, and bikes, while a summer "Adopt-A-Child" extension provides 300 children with backpacks, shoes, and school supplies. Despite its massive community impact, the organization has hit a growth ceiling due to archaic, manual administrative bottlenecks, such as volunteers spending 20 minutes per family transcribing handwritten, often Spanish-language intake forms into spreadsheets. 

### 2. BuildWithChrist.ai
BuildWithChrist.ai is an initiative by "City on Purpose" that hosts hackathons aimed at solving high-impact social problems in 4 hours using AI. Their March 7, 2026 event in Alpharetta, GA, is specifically dedicated to building automated systems for the Adopt-A-Family program. The live event features a broadcast-quality "show" format and has attracted national media attention, including an upcoming feature on EWTN news. The organizers actively encourage experienced developers to build parallel, open-source modules remotely to complement the in-person builder teams.

### 3. The Core System Built by the Main Team
The main hackathon teams used the Lovable platform to build a centralized, automated web application that eliminates manual spreadsheet data entry. The system is divided into five core modules: a bilingual Family Needs Intake form, a Sponsor Sign-Up portal, a Matching Console for coordinators, a Packet Builder & Checklist for gift logistics, and a Coordinator Dashboard. Under the hood, it uses a centralized database with a strict status state machine (`unmatched`, `matched`, `packet-sent`, `closeout`, `fulfilled`) to track the progress of `families` and `sponsors`. Crucially for future developers, the primary database tables feature a flexible `metadata` JSONB column designed to allow custom, extensible add-ons—like AI matchmaking or demographic reporting—without altering the core schema.
